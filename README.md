# calculator

线性代数小工具，大二学完线性代数的假期做着玩的。原本解题算法基本全手写，但是解复杂方程组的性能和精度太捉急，所以用科学计算库替换掉了几个轮子。现在还剩下一些线性变换的实现用于求矩阵的秩等。（科学计算库真是太牛逼了）

包含了向量计算器、单矩阵计算器、双矩阵计算器、 线性方程组求解器4个模块。

解方程组还算比较实用，可以帮小学生做作业（bushi）
