# calculator

线性代数小工具，大二学完线性代数的假期做着玩的。原本解题算法全部手写，但是解复杂方程组的精度有点捉急，后来用科学计算库替换掉了自己写的几个某几个轮子。

包含了向量计算器、单矩阵计算器、双矩阵计算器、 线性方程组求解器4个模块。
